Second, a .vcf file will be created once the backup process is complete. This file contains all of your contacts. You need to transfer this file to your Android phone. You can do so by emailing the file to yourself and downloading it on the Android device or using a USB cable to transfer the file.

Third, once the file is downloaded on your Android device, go to the Contacts app and click ‘Import.’ Here, you will be given the choice to select the .vcf file that you transferred.
