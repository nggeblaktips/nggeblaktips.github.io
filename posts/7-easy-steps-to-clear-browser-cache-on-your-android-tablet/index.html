1. Understand What Browser Cache Is#

Having trouble viewing webpages on your Android tablet? One of the leading causes can be what is known as browser cache. Browser cache is a collection of data stored on your device when you view a web page. This stored data allows the page to be loaded faster when you visit the same website again. If not cleared, the stored data can cause problems with the display of the pages. Knowing how to clear browser cache on your Android tablet can help you with these issues. Here are 7 simple steps to get you started:

  1. Open the browser and select the ‘More’ icon.
  2. From the bottom of the screen select ‘Settings’.
  3. Scroll down and select ‘Privacy’.
  4. Select ‘Clear Browsing Data’.
  5. Tick the ‘Cookies’ and ‘Cache’ box and select ‘Clear Data’.
  6. Restart the browser.
  7. Refresh any pages you have open.

By following these 7 simple steps, you can be sure that your browser cache is clear and you should be able to view pages normally. If you find you are still having trouble then you may need to look into other browser settings.
