1. How to do Screenshot on Windows 10 Laptop#

Taking screenshots on Microsoft laptops can be done in a few easy steps. You can capture images of your entire screen or just a single window. Windows 10 provides three methods to take screenshots, all of them easy to use.

First, you can take a screenshot with the keyboard. Press the Windows Logo Key + PrtScn keys together and the screen will dim for a moment indicating that the screenshot has been saved to the clipboard. You can paste the image into an image editor like Paint and save it.

Second, you can take a screenshot of the active window by pressing Alt + PrtScn keys together. This will capture the active window and save it to the clipboard. As with the previous method, you can paste the image, edit it, and save it.

Finally, you can use the built-in Snipping Tool to take more customized screenshots. To access it, open the Start Menu and search for Snipping Tool. You can select the type of screenshot you want, and even use a timer to delay capturing the screenshot. All screenshots taken with the Snipping Tool will open in a new window where you can save or copy the image.

With just a few simple steps, you can easily take screenshots on Microsoft laptops. By using keyboard shortcuts, the Snipping Tool, or a combination of both methods, you will always have a quick and easy way to capture images of your screen.
